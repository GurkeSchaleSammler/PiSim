# Life Simulation

Dies ist eine einfache Lebenssimulation, in der Tiere lernen, wann sie essen oder sich bewegen müssen, um zu überleben.

## Installation

1. Installiere die notwendigen Bibliotheken:
    ```
    pip install -r requirements.txt
    ```

2. Starte das Programm:
    ```
    python src/main.py
    ```
